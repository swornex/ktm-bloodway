# Installation

Step 1:

Open the folder in code editor.

Step 2:

Open the terminal

Step 3:

`yarn`

Step 4:

Create a .env file

Step 5:

Copy the content from .env.example to .env file

Step 6:

Fill the empty field in env file.

Step 7:

`yarn start:dev`

Step 8:

Open browser and hit http://localhost:3000
